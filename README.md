📖Booklet User
Aplikasi 'Dibuatin.CV' dilengkapi dengan dokumentasi panduan sistem yang lengkap untuk berbagai hak akses
Anda dapat membaca atau mengunduh buku panduanlengkapnya melalui tautan dibawah ini:

[Unduh Buku Panduan Pengguna Dibuatin.CV (PDF)](User_Guide_Dibuatin.CV.pdf)

📌Ringkasan Alur Sistem
1. **Hak Akses Pengguna**: Melakukan login, verifikasi keamanan kode OTP, memilih template CV, menginput formulir data diri, dan mengunduh hasil akhir dokumen berformat PDF setelah disetujui.
2. **Hak Akses Admin**: Memantau antrean berkas masuk, melakukan pengujian kecocokan tata letak data (*generate*), serta meneruskan berkas ke pihak konsultan.
3. **Hak Akses Konsultan**: Melakukan peninjauan mendalam pada lembar dokumen CV dan memberikan catatan evaluasi atau umpan balik (*note*) perbaikan.
